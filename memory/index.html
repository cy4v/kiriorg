暱稱 被愛麗絲踩著的石頭
稱號 SAO生還者
現在時間是「二零二零年五月三十日」，今天是我此生以來最悲痛的時刻，
為什麼呢？這都要從同年的五月五日開始說起，當時我正在瀏覽巴哈姆特論壇的「場外休憩區」，
當時我看見了一篇奇怪的文章，一篇掛上了星爆之名而熱度有著相當程度的文章；

這在當時是相當罕見的，當時只有大創作者的作品能夠擁有如此高的熱度，我抱著期待的心情點近了網頁中，
「https://mykirito.com/」，這就是我當時看見的網站，Kirito是動漫刀劍神域中的主角，
在場外常常出現並容易被針對式的按下倒讚，近乎已經成為了梗，但是這篇文章在當時只有少少的流量，
不過按讚數近乎過百，帶給了我相當大的衝擊，我坐在剛買的電腦前，疑惑的點開了網站，
點開網站後，看見的是需要Google登入的畫面，除此之外無他，正當我認為這是一個欺騙帳號的網站而想要離開時，
我的手卻已經不自覺的將帳號與密碼打上了畫面中，我被嚇呆了，面前是一名穿戴著黑色衣服的少年，
身後背著一把與衣服相襯的劍，正對著我微笑著，我使用滑鼠滑向下方，找到了「角色介紹」的按鈕，
我按下後尋找著剛剛那位黑衣少年，沒想到正坐落在最上方，「桐人」這是他的名子，以及簡單易瞭的技能解說，
不過下方有著？？？，這是什麼意思呢？我疑惑著，接著點回主頁面，又點開了角色列表的第一頁，我看見了另外一個「桐人」，
一樣卻又不一樣的角色，與我相異的是龐大的屬性與不同的角色立繪，他的圖片是正拔出雙刀的時期，不禁讓我想起了「星爆氣流斬」，
能否在這個遊戲中重現那一招？我想著，越來越興奮，我看著下方的行動，有著不同的六個按鈕，這六個按鈕想必有著完全不同的功能，
我按下了「坐下休息」的按鈕，並且獲得了15點經驗值，不過我並沒有升級，想必是需要更多經驗吧，按下後也隨之出現了冷卻時間，
大約60秒啊...，我想著，這六十秒可以做什麼事情呢？我又滑向了角色列表，看見與我等級相近的玩家們，不過已經有些人不再使用桐人，
改為的是其他女性角色，名子是...亞絲娜？我點進這位玩家的角色資訊中，下方又出現了四個按鈕，我選擇了最上層的「友好切磋」，
畫面上顯示著戰敗，我的神情隨之沮喪起來，我點回我的角色資訊，繼續點選坐下，我依然獲得了15經驗，不過這次我升級到了等級二，
多麼令人開心的事情，沒有其他事情能夠比起讓自己的角色成長更愉悅了，不知道創作者是誰啊...，真希望他是個好人呢！

思緒拉回了現在，「二零二零五月三十一日」，不知不覺，時間就已經到明天了呢，
我看著上方的轉生，想著有始有終的我最終選擇了尤吉歐，沒錯，就是尤吉歐，他跟著我打完了第二週目，成功得到了愛麗絲，
但當我開始玩愛麗絲之後就再無使用過他，我對此感到相當的愧疚，怎麼可以有了新歡就忘了舊愛？
我一巴掌拍在自己臉上，將點數洗白後點選了攻擊與技巧各一百的選擇上，並點下了「轉生」，這時候的轉生，無異於將一百等的機會抹殺，
但是我依舊選擇轉生回尤吉歐，沒有任何的廢話，沒有任何的遲疑，我只不過想在最後的日子裡，用最想要玩的角色，一舉通過所有的樓層，
我看著我的等級因為樓層獎勵而飛速飆漲，這些等級對我來說早已麻木，差別只在於87或者是100而已了，如今，奧斯頓與愛麗絲的開放讓所有的角色都不是其一合之敵，
我不懼怕死亡，我總認為遊戲角色的死亡比我現實世界的死亡更重要，但現在，在這個遊戲即將關閉之時，我感覺到了。我現在所使用的這隻角色，
跟我是一體的，如果他死了，我應該也會隨著他而去吧，我總祈禱著不會因為那「友好切磋」的極低機率而死亡，而這個遊戲也確實的回應我了，
無數次的友好切磋，都沒有停下我的腳步，我向著87等奔馳而去，當我終於抵達了頂點之前，我將樓層重置，回到了第一層，
也就是第一次遊玩這遊戲的惡夢「狗頭人之王伊格凡爾」之前，我點下了挑戰的按鈕，如今的伊格凡爾，早就敵不過脫胎換骨的我，
但對我來說，它永遠都是帶給我衝擊的BOSS之一，接著依然是伊格凡爾，只不過提前拔出了野太刀，不過仍舊不是我的敵手，
接著，第三層、第四層...一直到達了第七層為止，能夠轉變屬性的康妮，當初我在她手下吃了不少苦頭，她將我的攻擊與防禦互換，
但我被轉換過後的攻擊還是遠高於她的防禦，康妮被我斬於劍下。就這樣，一直一直推進，到達了讓眾玩家都吃盡苦頭的25層，也是我最大的難關，
據說到我打之前，也僅有兩位尤吉歐成功突破了這個難關，「他們可以，難道我就不行嗎？」這個想法在我的腦中迴響著，
我等待了五分鐘，最後還是點下了按鈕，出現了大量的戰報，我慢慢的、慢慢的往下滑，瀏覽著；我最後看見的是：康妮倒下了，我還剩下1點血量，
奇蹟真的發生了？我一時反應不過來，沒想到我真的能夠贏過第25層的康妮，也結束了我跟康妮們的孽緣，最大的難關已經跨越，剩下的關卡不再是問題，
最後了...真的是最後了嗎？我看著畫面中的閃耀魔眼，這個曾經帶給我絕望的敵人，最後也倒在血泊之中，我勝利了，我看著希茲克里夫...不對...是茅場晶彥，
真是非常精采的表演呢，他這麼說道，最後的最後，果然還是要有英雄對決魔王，不是嗎？我提劍奔上，使出了許久以前桐人教導我的水平扇形斬，
劍直直砍在希茲克里夫的盾牌之上，摩擦出陣陣火花，而他使用無人知曉的劍技，將我打倒了，血流不止的我，看到了一大群的玩家，我的心願已了，
希望這些玩家能夠擊敗茅場晶彥吧，語畢，我化為一把紅色的冰劍。

也是在那一個版本，我開始認真去遊玩這個遊戲，目標為突破最高樓層。不過我卻選擇了西莉卡這一個當時並不強勢的角色，原因很簡單，因為婆啊幹。我也在用西莉卡闖蕩艾恩葛朗特的途中，加入了DC的畢娜群，與友善的大家一起「鑽研」可愛的西莉卡。最終我也與群中的好夥伴流星，一起突破了當時最高的二層樓，也就是那兩個老害「奧斯頓與愛麗斯」，拿到了當時頗為稀有的愛麗絲。

後來，這個版本除了洗轉生點大量屠殺底層的問題，還有團長伺服器需要轉移等等，促使了這輪遊戲在十天內便要結束了。而當時的榜首智慧巨人殞落，和歐周仁的藍染被GVG追著砍，都是令我印象深刻的回憶。版本的最後，團長還有開放眾人圍毆希茲克利夫的小型收尾活動，不過我在睡覺沒有跟到就是了ㄏㄏ。

下一個版本，也是最後一個版本，應該是帶給所有人最少回憶的版本吧。畢竟開放修練，讓多數玩家都不需要再爆肝拚等級了，挑戰BOSS也是天方夜譚。然而BOSS強度上修，加上智慧這個能力值的削弱，讓西莉卡的地位比以往低了不少。因此，我的叛忍之路就此展開，養了一隻愛麗絲來爬塔。不過在爬到這個版本的BOSS MVP候選，也就是大家俗稱的破鳥時，可能是叛忍帶來的雖運，我打第一次破鳥就被送下去了QQ。不過在這段期間，看著前排的攻略之鬼帶著疊層一次又一次突破團長精心設計的BOSS們，還有團長那幾句名言 EX：「我建議亞絲娜就自己乖乖轉生下去，轉生並不可恥，不要挑戰正義的嗩吶」「亞絲娜這種種卑鄙的作弊角準備接受嗩吶的制裁」以及大家最喜歡的最後一句經典台詞：「怎麼可能會過20層阿媽的是不是作弊阿」
現在回想起來，還是覺得團長講的幹話特別好笑。

那麼還是真的很感謝團長，即使大家都說著2022年會出SAO

也許會有人說死宅炮中二病之類的，但有多少人懷抱著SAO的夢想，明白人也不少

我永遠不會忘記，這名為Mykirito，曾帶給多少人樂趣的遊戲。
如果還能有機會跟各位一起星爆，那就一定會再次拿起雙刀。
